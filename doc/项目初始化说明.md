# 项目初始化说明

## 目录结构

```text
myweb
├─ anime-log-server
├─ anime-log-web
├─ doc
└─ sql
```

## 后端

1. 执行 `sql/anime_log_db.sql`
2. 修改 `anime-log-server/src/main/resources/application.properties`
3. 用 IDEA 作为 Maven Web 项目导入
4. 挂到 Tomcat 运行

## 前端

1. 进入 `anime-log-web`
2. 执行 `npm install`
3. 执行 `npm run dev`

## 当前已完成

- SSM 后端基础目录和配置
- 用户、番剧、追番、单集日志基础接口骨架
- Vue 前端基础界面和路由
- 启动说明文档
